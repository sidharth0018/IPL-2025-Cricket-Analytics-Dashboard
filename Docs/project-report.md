# IPL 2025 Cricket Analytics Dashboard – Project Report

## Project Overview

The IPL 2025 Cricket Analytics Dashboard is an interactive data visualization project designed to analyze player and team performances throughout the IPL 2025 season. The dashboard provides insights into batting, bowling, team comparisons, performance trends, and strategic indicators using interactive charts and KPIs.

The objective of this project was to transform raw cricket statistics into meaningful visual insights that can support performance analysis and decision-making.

---

## Problem Statement

Cricket generates a large amount of statistical data, making it difficult to identify performance trends through raw numbers alone.

This project aims to:

* Analyze batting and bowling performances.
* Compare teams using key performance metrics.
* Identify top-performing players.
* Generate strategic insights from season statistics.
* Present data in an interactive and visually appealing format.

## Key Insights

* Sai Sudharsan emerged as the highest run-scorer of IPL 2025.
* Prasidh Krishna finished as the leading wicket-taker.
* Gujarat Titans demonstrated strong balance in both batting and bowling departments.
* Jasprit Bumrah recorded one of the best economy rates among frontline bowlers.
* Several teams displayed significant differences between batting strength and bowling effectiveness.
