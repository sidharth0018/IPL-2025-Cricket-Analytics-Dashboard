# IPL 2025 Cricket Analytics Dashboard

## Overview

An interactive cricket analytics dashboard built to analyze IPL 2025 player and team performances.

The dashboard provides insights into batting, bowling, team comparisons, KPI tracking, and strategic performance analysis through interactive visualizations.

## Features

- Batting Analytics
- Bowling Analytics
- Team Performance Comparison
- KPI Cards
- Interactive Charts
- Strategic Insights
- Responsive Dashboard Design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js
- GitHub Pages

## Dataset

Source: IPL 2025 Dataset from Kaggle

Dataset includes:
- Batting Statistics
- Bowling Statistics
- Team Performance Metrics
- Player Records
